# 🎬 Movie Library

A Java command-line app to manage a personal movie collection. Add, remove, view, and export your movies to a text file.

---

## Features

- View all movies in your library
- Add a movie (title, genre, year, sequel status)
- Remove a movie by index
- Clear the entire library
- Export your movie list to a `.txt` file

---

## How to Run

```bash
javac Movie.java
java Movie
```

---

## Example

```
========== COMMANDS ==========
1. View all movies
2. Add a movie
3. Remove a movie
4. Remove all movies
5. Export movies as .txt
6. Quit
```
